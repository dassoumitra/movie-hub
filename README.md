# 🎬 MovieHub

MovieHub is a movie discovery web application that allows users to explore trending movies, search films, view ratings, read movie details, and watch trailers.

The application uses The Movie Database API (TMDB) to fetch real-time movie data such as posters, ratings, overviews, and trailers.

## 🚀 Features

- 🔎 Search movies
- 🎬 Trending movies slider
- ⭐ Movie ratings
- 📄 Movie detail pages
- ▶ Trailer popup
- 🎭 Genre filtering
- 📑 Pagination
- ❤️ Watchlist using localStorage
- 📰 Movie articles and recommendations
- 📱 Responsive UI
- 💰 AdSense-ready structure

## 🛠 Tech Stack

- TMDB API

## ⚙️ Setup

1. Clone the repository

git clone https://github.com/yourusername/moviehub.git

2. Get an API key from TMDB

3. Replace the API key inside `script.js`

const API_KEY = "YOUR_API_KEY";

4. Open `index.html` in your browser.
